# pokedex 
PokeDex - Despliegue Seguro en Azure
Este proyecto consiste en el despliegue funcional y seguro de la aplicación PokeDex para la compañía Pueblo Paleta Inc., siguiendo prácticas profesionales de DevOps.

URL Pública
Acceso a la App: https://pokedex-smm-cgcggmfnasb0dggk.eastus-01.azurewebsites.net/

Creación de la Cuenta en la Nube
Para cumplir con el requerimiento de “Azure for Students”, se realizaron los siguientes pasos
Acceso: Se ingresó al portal de Azure Microsoft en la sección de beneficios académicos

Registro: Se utilizó el correo institucional y se completó el formulario con los datos de la institución educativa.
Seguridad: Microsoft solicitó validación de identidad mediante la aplicación Authenticator, la cual fue configurada y aceptada desde el dispositivo móvil para garantizar un acceso seguro.
